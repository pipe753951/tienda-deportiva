# Tienda deportiva

Este es un proyecto de Laravel sobre un sitio web básico de una tienda deportiva. Este sitio web fue creado junto con Tailwind CSS y Alpine.js

## Configuración del proyecto

Se requiere configurar la base de datos para probar y trabajar sobre el proyecto. Para configurar la base de datos para que el proyecto funcione, asegurate de que el archivo de entorno (```.env```) esté configurado para conectarse correctamente a la base de datos, y luego, realiza una migración, y ejecuta los seeders.

```bash
php artisan migrate
php artisan db:seed
```
